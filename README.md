# fg_locationhud

**German:**  
Ein benutzerfreundliches HUD für FiveM, das in Echtzeit anzeigt, an welchem Standort du dich gerade befindest. Dieses HUD bietet eine klare und intuitive Möglichkeit, deine aktuelle Position innerhalb der Spielwelt zu verfolgen, sodass du dich besser orientieren kannst. Ideal für Roleplay-Server oder andere FiveM-Projekte, bei denen Standortinformationen wichtig sind.  

**English:**  
A user-friendly HUD for FiveM that displays your current location in real time. This HUD provides a clear and intuitive way to track your position within the game world, helping you navigate more efficiently. It is ideal for roleplay servers or other FiveM projects where location awareness is important.
